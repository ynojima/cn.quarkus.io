文件中使用的变量

以下变量外部化了可能随时间而变化的关键信息，因此对这些信息的引用应该通过使用大括号{}中的变量来完成。下表给出了可使用的外部化变量的完整列表。

Table 1. 变量
Property Name Value 描述

{quarkus-version}

2.7.5.Final

项目的当前版本。

{quarkus-hom-url}

{quarkus-hom-url}

项目主页的位置。

{quarkus-site-getting-started}

/get-started

入门页面的位置。

{quarkus-org-url}

https://github.com/quarkusio

The location of the project github organization.

{quarkus-base-url}

https://github.com/quarkusio/quarkus

Quarkus GitHub URL的通用基础前缀。

{quarkus-clone-url}

https://github.com/quarkusio/quarkus.git

文档所引用的git clone的Quarkus URL。

{quarkus-archive-url}

https://github.com/quarkusio/quarkus/archive/master.zip

指向主源存档的Quarkus URL。

{quarkus-blob-url}

{quarkus-blob-url}

指向主blob源树的Quarkus URL；用于引用源文件。

{quarkus-tree-url}

https://github.com/quarkusio/quarkus/tree/main

指向主源树根的Quarkus URL；用于引用目录。

{quarkus-issues-url}

https://github.com/quarkusio/quarkus/issues

指向问题页的Quarkus URL。

{quarkus-images-url}

https://github.com/quarkusio/quarkus-images/tree

指向提供给Quarkus的容器镜像集的Quarkus URL。

{quarkus-chat-url}

https://quarkusio.zulipchat.com

我们聊天的URL。

{quarkus-mailing-list-subscription-email}

quarkus-dev+subscribe@googlegroups.com

用于订阅我们邮件列表的电子邮件。

{quarkus-mailing-list-index}

https://groups.google.com/d/forum/quarkus-dev

邮件列表索引页。

{quickstarts-base-url}

https://github.com/quarkusio/quarkus-quickstarts

Quickstarts URL 通用基础前缀。

{quickstarts-clone-url}

https://github.com/quarkusio/quarkus-quickstarts.git

文档所引用的git clone的Quickstarts URL。

{quickstarts-archive-url}

https://github.com/quarkusio/quarkus-quickstarts/archive/main.zip

指向主源存档的Quickstarts URL

{quickstarts-blob-url}

https://github.com/quarkusio/quarkus-quickstarts/blob/main

指向主blob源树的Quickstarts URL；用于引用源文件。

{quickstarts-tree-url}

https://github.com/quarkusio/quarkus-quickstarts/tree/main

指向主源树根的Quickstarts URL；用于引用目录。

{graalvm-version}

21.3.1

推荐使用的GraalVM版本。

{graalvm-flavor}

21.3.1-java11

要使用的GraalVM的完整版本，如“19.3.1-java11”。请确保使用 java11 版本。
